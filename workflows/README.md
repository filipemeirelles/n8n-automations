# Workflows n8n

Cada subpasta contém:
- `README.md` — descrição, fluxo, configuração
- `workflow.json` — export do workflow n8n (quando finalizado)

## Como adicionar um workflow

1. Desenvolva no n8n
2. Exporte: Menu (⋮) → Download
3. Crie pasta `workflows/NN-nome-do-workflow/`
4. Adicione o JSON e um README.md descritivo
5. Commit e push
