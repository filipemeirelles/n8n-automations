# Assets

Screenshots, diagrams e imagens dos workflows.

## Screenshots recomendados

Para cada workflow, capture:
1. **Visão geral** — o canvas completo do workflow
2. **Nodes detalhados** — configurações importantes
3. **Output** — exemplo de execução funcional

## Formato

- `.png` ou `.webp` (melhor compressão)
- Nome: `workflow-nome-screenshot.png`
